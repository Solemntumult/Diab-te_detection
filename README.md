# Diabete_detection
Modèle de détection de diabète en tenant compte du Nombre de Grossesses Glucose (mg/dL), de la Pression Artérielle (mm Hg), de l'Épaisseur Peau, du taux d'Insuline (mu U/ml), de l'IMC, de la Fonction Pedigree et de l'âge. Basé sur un Dataset de 768 patients, le modèle est entrainé sur 80%(615 patients) et testé sur les 20% restant.
